GINICredentialsStore Protocol Reference

Conforms to NSObject
Declared in GINICredentialsStore.h

Overview

The GINICredentialsStore defines the methods required to store and fetch user-related data. This can be refresh token when the server-side OAuth authorization flow is used or the actual user data when the “anonymous” user feature is used.
