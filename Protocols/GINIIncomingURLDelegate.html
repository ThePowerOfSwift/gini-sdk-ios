GINIIncomingURLDelegate Protocol Reference

Conforms to NSObject
Declared in GINIIncomingURLDelegate.h

Overview

This protocol is used for checking if a specific URL should be handled by the app.

It links the part where the ApplicationDelegate receives a new incoming URL and needs to pass it to the session manager to check if it is a authentication request.
