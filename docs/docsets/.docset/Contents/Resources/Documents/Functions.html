  • GINIInjectorKey

    Helper function to use an an arbitrary Objective C object in dictionaries that describe or provide dependencies. Wraps the given object into a NSValue* if it does not conform to the protocol, otherwise returns the same object. The returned value can always be used as the key for the given object in a NSDictionary, e.g. in a mapping when providing dependencies.
