GINIFactoryDescription

@interface GINIFactoryDescription : NSObject

A value object for storing information about a factory. Used by the GINIInjector.

  • isSingleton

    Whether or not the created instance is thought as a singleton. Please notice that a singleton is a singleton per injector, not per application.

  • dependencies

    An array containing dependency keys. Instances of the object for the given key are given as arguments to the factory in the order they are appearing in this array.

    A selector for the method on the class or object.
