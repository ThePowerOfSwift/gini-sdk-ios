GINIIncomingURLDelegate

@protocol GINIIncomingURLDelegate <NSObject>

This protocol is used for checking if a specific URL should be handled by the app.

It links the part where the ApplicationDelegate receives a new incoming URL and needs to pass it to the session manager to check if it is a authorisation request.

  • -handleURL:

    Checks if a specific URL should be handled by the implementing class of this method, it should return YES if is handled by the class or NO otherwise.
