Overview

The GINIURLResponse is a value object for the result of an HTTP request. It is used inside the Gini SDK to have the possibility to pass the interpreted result of an HTTP communication (the data property) together with the HTTP request’s original meta data (like HTTP headers and so on).

Usually you don’t use this data object directly. It is only used by low level classes such as the GINIURLSession class.
