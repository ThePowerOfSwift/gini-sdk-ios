previewWithSize:forPage:

Gets the preview image for the given page.

- (BFTask *)previewWithSize:(GiniApiPreviewSize)size forPage:(NSUInteger)page

Parameters

size

The size of the rendered preview. Please notice that those sizes are the maximum sizes of the renderings, the actual image can have smaller dimensions.

page

The page for which the preview is rendered. Please notice that only the first 10 pages of a document are processed by the Gini API.

Discussion

Gets the preview image for the given page.
