initWithURLSession:requestFactory:baseURL:

The designated initializer.

- (instancetype)initWithURLSession:(id<GINIURLSession>)urlSession requestFactory:(id<GINIAPIManagerRequestFactory>)requestFactory baseURL:(NSURL *)baseURL

Parameters

urlSession

An object that implements the GINIURLSession protocol. Will be used to perform the HTTP communication.

requestFactory

An object that implements the GINIAPIManagerRequestFactory. Will be used to create request objects with valid session credentials. @see GINIAPIManagerRequestFactory for details.

baseURL

The baseURL. The requests to the Gini API are relative to that URL, so it usually should be set to a NSURL* pointing to ‘https://api.gini.net’.

Discussion

The designated initializer.
